# Inventory Analysis Dashboard

## Project Overview
This project focuses on analyzing inventory data to monitor stock levels, evaluate product performance, and optimize inventory management decisions.

## Tools Used
* Microsoft Excel (Data Cleaning & Preparation)
* Power BI (Dashboard & Visualization)

## Dataset
The dataset includes:
* Product Name
* Category
* Stock Quantity
* Sales Quantity
* Purchase Date
* Supplier
* Price
* 
## Key Metrics
* Total Revenue = Price × Sales Quantity
* Inventory Value = Price × Stock Quantity

## Dashboard Features
* KPI cards (Total Revenue, Inventory Value, Total Stock)
* Stock by Category
* Revenue by Category
* Supplier Performance
* Monthly Purchase Trend
* Top 5 Products by Revenue
* Interactive slicers (Category, Supplier, Month)
  
## Key Insights
* High-value products contribute significantly to total revenue
* Some products show overstock risk (high stock, low sales)
* Some products indicate stockout risk (low stock, high sales)
* Supplier analysis helps identify key vendors
